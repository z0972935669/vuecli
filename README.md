# 安裝
========================================
Node版本要求: 使用nvm, 推薦更高版本

使用以下指令進行安裝
----------------------------------------
npm install -g @vue/cli
#or
yarn global add @vue/cli

安裝完後，可以在命令行中訪問vue版本，驗證是否安裝完成。
----------------------------------------
vue --version
#or
vue -V

創建一個項目
========================================
vue create [project Name]
